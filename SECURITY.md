# SECURITY.md

## Security policy

If you believe you have found a security issue in this repository (for example, a vulnerability in the script logic that could cause unsafe behavior, or sensitive data accidentally committed), please report it responsibly.

## Reporting

- Open an issue only if the report contains **no sensitive details**.
- If the report includes sensitive details, do not post it publicly. Instead, contact the repository owner privately (for example via GitHub profile contact).

## Scope

This repo contains defensive scripts for detection/removal of a specific malware persistence pattern. It is not intended for offensive use.
